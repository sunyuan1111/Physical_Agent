## Driver Contract

Two files are enough to connect a device to Physical Agent:

```text
physical_driver.yaml
driver.py
```

The manifest declares the adapter and validates configuration.
The driver implements `PhysicalDriver` and stays on the watch side.

## How to Finish the Integration

1. Keep the driver logic on the watch side.
2. Wire the endpoint, token, and timeout settings into physical_driver.yaml.
3. Move hardware SDK calls into driver.py.
4. Run mock mode first, then switch to the real bridge or device.
5. Add a focused pytest for each capability you keep.
6. Keep pick and place actions dependent on the gripper state.
7. If the repo already has an integration script, copy the transport logic, not the README wording.
